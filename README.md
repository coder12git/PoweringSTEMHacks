# PoweringSTEMHacks

## Running

1. Create a virtual environment

    ```bash
    python3 -m venv venv
    ```

1. Activate the virtual environment

    for Linux and Mac:

    ```bash
    source venv/bin/activate
    ```

    for Windows:

    ```bash
    venv\Scripts\activate
    ```

1. Install dependencies

    ```bash
    pip install -r requirements.txt
    ```

1. Run the app

    ```bash
    streamlit run ./About.py
    ```
